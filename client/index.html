URL Saver

✓ App is running in the system tray

Shortcut: Press Cmd/Ctrl + Shift + S to save URLs or images

📌 First Time Setup - Enable Browser Extensions

To save images by hovering, enable the browser extension:

  1. Chrome/Arc/Edge: Go to chrome://extensions (or arc://extensions / edge://extensions)
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked"
  4. Select the extension folder:
    Title: chromium
    Path: [Project Root]/browser-extension/chromium
  5. Extension will now work automatically!

💡 This is a one-time setup. After enabling, it works automatically.
