Cmd+C로 이미지와 SNS 게시물을 저장하는 Chrome 확장 프로그램

Chrome extension to save images and social media posts with Cmd+C

주요 기능 / Features

  • ⚡ 키보드 단축키로 즉시 저장 / One-key save with keyboard shortcut
  • 📎 클립한 이미지를 클립보드로 바로 복사 / Copy a clipped image straight to your clipboard
  • 📁 로컬 폴더 또는 Google Drive에 저장 / Save to local folder or Google Drive
  • 🏷️ Img / SNS 카테고리로 자동 분류 / Auto-categorized into Img / SNS
  • 📋 사이드 패널에서 카드 형태로 관리 / Browse saved items in side panel
  • 🔒 개인정보 우선 · 오픈 소스 / Privacy-first · Open source
