# KickClip 설치 및 사용 가이드

KickClip 은 웹페이지 콘텐츠를 쉽고 빠르게 저장 + 복사할 수 있는 Chrome 확장 프로그램입니다! 본 가이드는 KickClip 의 설치 및 사용 방법을 안내합니다.

---

## 📦 설치 방법

### 1. ZIP 파일 압축 해제

받으신 `kickclip-v1.0.0-prod.zip` 파일을 임의의 위치에 압축 해제합니다.

압축을 풀면 다음과 같은 내용이 보입니다:

- `README.md` (이 문서)
- `kickclip-prod.zip` ← **이 파일을 Chrome 에 설치합니다**

### 2. kickclip-prod.zip 압축 해제

`kickclip-prod.zip` 도 압축을 해제하면 `kickclip-prod/` 폴더가 생성됩니다. 이 폴더를 **삭제하지 말고 고정된 위치**에 보관해주세요 (예: `~/Documents/KickClip/` 또는 `~/Applications/KickClip/`).

> ⚠️ **중요**: 이 폴더를 삭제하거나 이동하면 확장 프로그램이 작동하지 않습니다.

### 3. Chrome 확장 설치

1. Chrome 브라우저에서 주소창에 다음 입력: `chrome://extensions`
2. 우측 상단의 **"개발자 모드"** 토글을 **켭니다**
3. 좌측 상단의 **"압축해제된 확장 프로그램을 로드"** 버튼 클릭
4. 앞서 압축 해제했던 `kickclip-prod/` 폴더를 선택
5. KickClip 확장 프로그램이 목록에 추가되면 완료

### 4. 확장 고정 (권장)

1. Chrome 우측 상단의 **퍼즐 조각 아이콘** 클릭
2. 목록에서 **KickClip** 옆의 **핀 아이콘** 클릭
3. 주소창 오른쪽에 KickClip 아이콘이 고정됨

## 🚀 첫 실행

### 1. Side Panel 열기

새 탭을 열고 아무 주소 (예: google.com) 로 이동합니다. 왼쪽 위에 **"click [단축키] to start KickClip"** 형태의 안내 메시지가 표시되면 확장이 정상 로드된 것입니다.
(단축키는 OS 에 따라 `⇧⌘S` 또는 `Ctrl+Shift+S` 로 표시됩니다.)

다음 중 **편한 방법**으로 Side Panel 을 여세요:

- **단축키 사용**: `Cmd+Shift+S` (macOS) / `Ctrl+Shift+S` (Windows, Linux)
- **아이콘 우클릭**: 확장을 고정한 경우, 주소창 오른쪽의 **KickClip 아이콘을 우클릭 → "측면 패널 열기"** 클릭

### 2. 계정 로그인

처음 열면 로그인 화면이 표시됩니다:

- **Google 계정으로 로그인**

로그인 후 Side Panel 에 내 저장 목록이 표시됩니다.

> ⚠️ **베타 테스터 안내**: 아직 테스트 사용자 이메일 등록이 안 되어 있는 경우 로그인이 실패할 수 있습니다. 그럴 경우 `tlstlsgns@gmail.com` 으로 연락 부탁드립니다.

## 💡 기본 사용법

### 페이지 전체 저장

1. 저장하려는 웹페이지를 엽니다
2. 마우스를 페이지 위로 이동하면 **페이지 전체 테두리**와 **"Save It!" 배지**가 표시됩니다
3. 단축키 (`Cmd+Shift+S` 또는 `Ctrl+Shift+S`) 를 누르거나 배지를 클릭
4. Shutter 효과와 함께 저장 완료

### 특정 요소(CoreItem) 저장

1. 웹페이지의 저장하려는 요소 (이미지, SNS 게시물, 동영상 등) 위로 마우스를 이동
2. 요소 주변에 **보라색 테두리 + "Save It!" 배지**가 나타납니다
3. 배지 클릭 또는 단축키로 저장

### 저장 시 자동 클립보드 복사

저장 시 자동으로 클립보드에도 복사됩니다:

- **이미지 카테고리**: 이미지 자체가 클립보드에 복사됨 (`Cmd+V` 로 이미지 붙여넣기 가능)
- **SNS / 메일 / 페이지 카테고리**: URL 이 클립보드에 복사됨

저장 및 복사 결과는 화면 상단 중앙의 **Toast 알림**으로 확인할 수 있습니다.

### Side Panel 탭 전환

Side Panel 상단의 탭으로 카테고리별 저장 목록을 볼 수 있습니다:

- **Img**: 저장한 이미지들
- **SNS**: SNS 게시물 (Instagram, X/Twitter 등)
- **Mail**: 저장한 메일 링크
- **Pages**: 일반 웹페이지

## ⌨️ 단축키

### 기본 단축키

- **저장 및 Side Panel 열기**: `Cmd+Shift+S` (macOS) / `Ctrl+Shift+S` (Windows, Linux)

### 단축키 변경 방법

1. Chrome 주소창에 입력: `chrome://extensions/shortcuts`
2. KickClip 항목을 찾습니다
3. 해당 커맨드 옆의 연필 아이콘 또는 입력란을 클릭
4. 원하는 단축키 조합을 키보드로 입력
5. 자동 저장됨

> 💡 **팁**: 다른 확장 프로그램이나 브라우저 기본 단축키와 겹치지 않도록 주의하세요.
